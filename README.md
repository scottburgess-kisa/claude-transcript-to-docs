# transcript-to-docs

Turn a meeting transcript into a blog post, Slack update, and success story using Claude Code.

## What It Does

This workflow takes a transcript from a team discussion about a project and guides you through a series of stages to produce polished output documents. It asks clarifying questions along the way to fill gaps and tailor the content before generating anything.

## Getting Started

See the [Setup Guide](docs/setup-guide.md) for full instructions, including prerequisites and how to run the workflow.

If you are running the meeting that generates the transcript, see the [Facilitator Guide](docs/facilitator-guide.md) for guidance on how to structure the conversation to get the best results.

## Output Documents

- Blog post (LinkedIn format)
- Internal Slack update
- External success story
